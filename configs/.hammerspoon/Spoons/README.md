# Hammerspoon Scripts

Lua code to control
[:octocat: Hammerspoon](https://github.com/Hammerspoon/hammerspoon)
behaviour for system event handlers for
[macOS](https://github.com/HariSekhon/Knowledge-Base/blob/main/mac.md), such as:

- auto-switching to dual sound output when connecting AirPods to be able to Shazam songs from Love Island
